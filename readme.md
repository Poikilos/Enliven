# Enliven

Enliven is a game that focuses on immersion, through less cognitive load and more connection between you and the game world.

Reduced cognitive load means less memorization and more fun:
- Consistent interface and world behavior.
- Focusing on items and other features that add to gameplay, rather than piles of items that fill your inventory.
- Emergent behaviors instead of more systems to remember.

Emergent means combining things to make more than the sum of their parts. Making items and creatures in the world more reactive with each other and have more uses means less to learn and more to discover by playing around.

Stay tuned for progress toward the goals!

Roadmap and other info: [minetest.io](https://minetest.io)


## Where is it?
build-enliven.py gets the latest version of mods from official repos, or new repos where changes were necessary.

Releases and engines before the Luanti 5 API are no longer supported.
